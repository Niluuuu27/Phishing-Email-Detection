# Phishing Email Detection Model

A machine learning project that classifies emails as Phishing or Safe using Scikit-learn.

## Features

* Email text analysis
* Phishing email detection
* Safe email classification
* Accuracy calculation
* Confusion matrix generation

## Technologies Used

* Python
* Pandas
* Scikit-learn

## Files

* phishing_detector.py
* emails.csv
* result.txt
* README.md

## Output

The model predicts whether an email is:

* Phishing
* Safe

and generates accuracy and confusion matrix results.
